| 🧶 面料 Fabrics         | 透视 & 蕾丝、破坏 / 腐蚀感、工装面料、PU & Snake、灯芯绒 & 丝绒、3D 结构面料、绗缝、金属银、缎面       | ⭐⭐⭐⭐ | “派对 + 日常混合”，银色金属 & 高级缎很重要 |
| 🖼 印花 Prints          | 经典动物纹（豹 / 斑马 / 蛇）、柔和迷彩、侵蚀感花、抽象模糊、冬季扎染、波点、升级花卉、暗底亮花、拼接印花           | ⭐⭐⭐  | 动物纹“回潮”，波点升级为**趋势印花**     |

## 5. 面料趋势 Fabrics

### 5.1 透视 / 腐蚀 / 结构

| 面料方向                  | 说明           | 关键符号            |
| --------------------- | ------------ | --------------- |
| 👻 Sheer Fabrics      | 大量透视，派对季必备   | ⭐               |
| 🕸 Lace               | 作为细节或主体面料    | 与 Sheer 同步      |
| 🧪 Eroded / Destroyed | 像在地里埋了几年、腐蚀感 | 属于解构语系          |
| 🧱 3D Structures      | 有立体纹路的结构面料   | 延续 Y2K / 90s 感觉 |

### 5.2 功能 & 经典冬季素材

| 面料方向                   | 说明             |
| ---------------------- | -------------- |
| 🧰 Cargo Fabrics       | 坚韧工装面料，仍是焦点    |
| 🧴 PU / “Leather-look” | 秀场皮感极多，PU 必不可少 |
| 🐍 Snake Pattern on PU | 蛇纹尤其适合用在 PU 上  |
| 🧵 Corduroy & Velvet   | 柔软、适合秋冬色彩表现    |
| 🧷 Quilted Fabrics     | 绗缝=冬季基本盘       |

### 5.3 派对 &华丽感

| 面料方向               | 说明                    | 要点           |
| ------------------ | --------------------- | ------------ |
| 🪙 Metallics (银为主) | 从黄金转向**银色**焦点         | 任何银涂层、亮片都加分  |
| 🥂 Satin           | 继续重要，需**高级感，不要廉价涤纶感** | ⭐ 建议重点开发高档手感 |
| 🎉 Party Fabrics   | 各类闪片、亮片、派对材质          | 对应“派对季节”定位   |
