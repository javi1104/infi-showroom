IQC / AQC 考试一页速记

专供 12 月下旬理论考核临考突击 · 建议打印或手机收藏
模式：速记 / 核心考点
用途：考试前 5–15 分钟快速过一遍
考试基本信息 & AQC 身份
先搞清“我是谁”
考试 & 认证
  • 时间：12 月下旬（约 24 号）
  • 形式：理论题 + 实操相关理解
  • 通过条件：理论 + 实操均达标 → 颁发 AQC 证书
  • 没上过第一次培训的人：必须补课，否则考试很难通过
AQC 的真实身份
  • 一句话：AQC 代表 Bestseller 质量团队进行最终产品验收
  • 同时对两方负责：
  • ① 自己公司领导（外贸公司）
  • ② BS 各区域 QA 同事（质量团队）
AQC 评估（别以为拿证就完事）
  • 系统对比：AQC 自检 vs BSQC 复检
  • 看通过率、缺点数量 & 类型是否接近
  • QA 现场抽查：抽大货 → 看实物 → 对比自检报告
  • 目标：AQC 眼光 ≈ BSQC 眼光

记忆口诀：“代表 BS 验货 + 同时汇报公司领导和 BS QA + 数据说话”

缺点三大类（必考高频）
Critical / Major / Minor
严重缺点
Critical
  • 可能影响整批货销售
  • 或对消费者有 安全 / 健康风险
  • 例：断针、发霉、有明显霉味
可接受数量：0 个
主要缺点
Major
  • 导致单件衣服 无法销售 / 无法正常使用
  • 例：乒乓球大小破洞、大面积色差等
  • 注意：大破洞 = Major，不是 Critical
按 AQL 有限量接受
次要缺点
Minor
  • 消费者仍可能购买，但 不满意
  • 例：不显眼部位小线头、小脏污
按 AQL 有限量接受

必背句：严重 = 安全 / 整批风险；主要 = 单件不能卖；次要 = 勉强能卖但难看。
并且：Major 和 Minor 在 BS 系统里分开计数，不合并。

尺子校准 · 尺寸测量 · AQL 要点
抽样 & 标准最爱考
尺子校准（量错 = 白量）
  • 校准周期：约 3 个月一次
  • 示例误差：20cm ≤ ±1mm；50cm ≤ ±1.5mm；100cm ≤ ±2mm
  • 不仅 AQC 的尺子要校，要提醒工厂尺子一起校准
尺寸测量原则
  • 所有服装必须做尺寸测量
  • 混色检：每个尺码至少 3 件 以上
  • 发现某色 / 某码超标 → 该码该色额外多量 10–15 件
  • 不同品牌（如 Jack & Jones / JJXX vs 其他品牌）量法不同
AQL 抽样核心
概念 说明
抽样数量 同批货在 AQL 2.5 / 1.5 下 抽样件数相同
AQL 不同 AQL 越小 → 标准越严 → 可接受缺点数更少
示例 10,001 件 → 抽 315 件（AQL 2.5）
Major ≤14 个可接受
Minor ≤14 个可接受（单独计数）
Critical 1 个都不能接受

Jack & Jones / ONLY：
首检：AQL 2.5；复验：AQL 1.5（尺寸 & 做工同理）。

防霉 / 湿度 · 验针 · 物理测试
易丢分的小题
防霉 & 湿度
  • 缅甸 / 柬埔寨工厂：必须有 干燥房
  • 干燥房容量要与产能匹配
  • 湿度要求：＜ 50%（雨季尤其要控制）
  • 干燥剂品牌：Macropack
  • 真皮 / PU 等高风险：优先用 防霉纸
验针（减针）
  • 所有产品必须能通过 1.2mm 减针测试
  • 验针机要定期用标准测试片校准
  • 若无法过减针：必须提前报给 QA / 买手
  • 严禁：工厂私下决定不过减针、要求 AQC 不写 → 被 BSQC 抓住后，货 & AQC 都受影响
物理测试
  • 报告格式可用工厂 / 第三方模板
  • AQC 至少要看得懂：哪些项目合格 / 不合格
  • 面料克重常见公差：约 ±5%

记住一句：发霉 & 断针 = 巨额索赔，宁可多问 QA，也不要自己“悄悄决定”。

检验前资料清单（Checklist）
不齐这几样 ≈ 不该开工
必备资料（背下来）
  • 最新 原始订单（款号 / 订单号 / 数量 / 包装方式 / 交期 / 成分等）
  • 最新 尺寸表
  • 买手 样衣意见（Comments）
  • 确认样 + 锁扣（BS 锁扣或公司锁扣，不可拆）
  • 面料卡（正反面清楚，签名 / 盖章）
  • 辅料卡（辅料齐全，中英文对照，签名 / 盖章）
  • 物理测试报告：缩水、色牢度、克重等
  • 如资料拿不到 → 可私下找 BS QA 帮忙催业务

口诀：“订单 + 尺寸 + 样衣 + 锁扣 + 面辅料卡 + 测试报告” 一个都不能乱。

抽箱 · 包装 · 箱规
决定你“眼光好不好”的部分
抽箱原则
  • AQC 必须亲自随机抽箱，不能让工厂代抽
  • 每个颜色 × 每个尺码 都要被抽到
  • 数量多的尺码 / 颜色 → 多抽；数量少的 → 确保有样本即可
  • 配比包装：按箱数比例抽；独码包装：中码多抽，小 / 大码适量
  • 发现箱子破损：报告中备注并要求工厂更换，比例高时要查原因
包装 / 纸箱要点
  • 按形态分：挂装 / 箱装
  • 箱装再分：Loose（散装） / Assort（配比包装）
  • 基本箱规：L ＞ W ≥ H，有最小 / 最大尺寸要求（以包装手册为准）
  • Loose 订单必须有 天地板，防止丹麦仓库机器开箱割伤衣物
  • 相贴纸：正面 + 侧面各一张，位置可略有偏移，只要 方便扫码 即可
  • 箱重限制：Loose / 配比最大重量不同（手册中一般 Loose ≈ 23kg 左右）

重要：箱规为衣服服务，不要为塞进箱子硬把西装 / 毛呢折成“粽子”导致皱痕索赔。

产前会 · 人台拍照 · 报告结论
常见细节题
产前会（PP Meeting）
  • 参会：生产相关管理人员 + 工厂 QC + 外贸业务 / 跟单 + 必要时 BS QA
  • 两个核心目的：
  • ① 回顾工厂生产前所有准备工作是否到位
  • ② 明确质量要求：外观 / 做工 / 尺寸 / 包装 / 验针 / 测试等
人台 / 模特拍照
  • 外套类：人台内应穿一件薄衣（T 恤 / 衬衫），避免“光膀子 + 外套”
  • Jack & Jones / JJXX：必须用 指定人台，没有则写“不适用”，不要用其他人台代替
  • 其他品牌：有 BS 人台就用；没有可以不拍人台图，但别“装聪明”全部不拍
自检报告结论
  • 系统（手机）中结论只有两种：“合格 / 不合格”，无“待定”选项
  • 提交前务必检查：尺寸 / 照片 / 缺点记录是否准确
  • 一旦提交，报告无法撤回
自检全流程一行记住
一眼看懂“正确姿势”
1 接单 & 向业务要齐全资料
2 核对订单 / 尺寸表 / 样衣 + 锁扣 / 面辅料卡 / 测试报告
3 检前准备：校准尺子 & 验针机，确认包装方式 / 箱规
4 AQC 亲自按比例随机抽箱，覆盖所有色码
5 做工 + 尺寸 + 验针 + 湿度 / 防霉 + 称重
6 记录缺点（Critical / Major / Minor 分开记）并拍清楚照片
7 按 AQL 判定合格 / 不合格 → 系统提交报告
8 重要问题同步：工厂 + 自家领导 + 对应 BS QA

真正的“及格线”：会用这套流程 + 知道何时主动找 QA 沟通 → 才算合格 AQC。
