## 一、培训 & 考试信息总览

| 项目   | 内容                                      |
| ---- | --------------------------------------- |
| 培训类型 | IQC / AQC 第二次理论培训（偏“实操原理总结”）            |
| 培训目的 | ①系统梳理日常自检相关知识；②为 12 月下旬 IQC/AQC 考核做准备   |
| 培训时长 | 约 1 小时（实际超时约 20 分钟）                     |
| 参会人员 | 各外贸公司 IQC/AQC 候选人（国内、柬埔寨、缅甸等）           |
| 特别情况 | 有若干人**没参加第一次理论培训** → 后面安排补课，否则考试几乎不可能通过 |
| 考试时间 | 预计 12 月 24 日左右（12 月下旬），线上统一考试           |
| 考试通过 | 理论 + 实操都达标 → 颁发 AQC 证书/认证               |

**★关键记忆点：**

* 考前要**把两次培训内容 + 质量/包装/安全等手册翻一遍**。
* 没上过第一次理论培训的，会**单独再补一次**，否则基础概念缺失。

## 二、IQC / AQC 的定位 & 汇报关系

### 1. 岗位本质（一句话版）

> **IQC / AQC = 代表 Bestseller（贝斯勒/白色）质量团队进行最终产品验收的“授权检验员”**

### 2. 职责结构表

| 维度   | 说明                                          |
| ---- | ------------------------------------------- |
| 身份   | 受外贸公司雇佣，同时通过 Bestseller 培训 + 考核 + 认证的 AQC   |
| 核心职责 | 按品牌标准，对大货进行自检，代表 BS 质量团队做“出货前质量把关”          |
| 工作目标 | 和工厂 QC / 外贸公司 / BS QA 的目标一致：**减少出货问题、避免索赔** |
| 汇报对象 | ✅ 自己公司领导（外贸公司）<br>✅ BS 各区域 QA 同事（质量团队）      |

> 🧩 **要点：不是“只听谁”的问题，而是“同时对双方负责”。**

**★关键记忆点：**

* 把自己当成“**Bestseller 质量部派驻在工厂的检验员**”。
* 有问题要**同时**和：①自己公司领导，②对应 BS QA 保持紧密沟通。

## 三、AQC 的评估机制（不是拿证就万事大吉）

| 评估方式      | 具体做法                                           | 目的                  |
| --------- | ---------------------------------------------- | ------------------- |
| ① 结果数据对比  | 系统中对比同一批货：AQC 自检结果 vs BSQC 复检结果（通过率、缺点比例、缺点类型） | 看 AQC 眼光是否和 BSQC 接近 |
| ② QA 现场抽查 | QA 临时到工厂：抽大货 → 看实物 → 对比 AQC 自检报告               | 检查记录是否真实/完整/一致      |

* 评估频率：**定期 + 不定期**（大约每半年 QA 心里都会有“账本”）
* 结果处理：

  * 表现好：继续合作，自检比例有机会提高，BSQC 抽验减少
  * 表现差且不改进：会沟通、辅导，严重则考虑**更换 AQC**

**★关键记忆点：**

* **“跟 QA 关系好不好”不是评估标准，评估靠系统数据 & 实物对比。**
* 目标：**AQC 的判定结果尽量接近 BSQC**（发现问题的类型/比例接近）。

## 四、产品缺点（Defects）三大分类

### 1. 分类表

| 类型         | 英文       | 典型特点                           | 示例                  | 验收要求                  |
| ---------- | -------- | ------------------------------ | ------------------- | --------------------- |
| ⚠ **严重缺点** | Critical | ① 可能影响整批货销售<br>② 或对消费者有安全/健康风险 | 断针、发霉、有明显霉味、严重安全问题等 | **0 个可接受**（发现 1 个即严重） |
| ◆ **主要缺点** | Major    | 导致 **单件产品无法销售/无法正常使用**         | 一件衣服有一个乒乓球大小的破洞     | 允许数量有限（AQL 限值）        |
| △ **次要缺点** | Minor    | 消费者还会购买，但**不满意**，影响外观/感受       | 不显眼部位的小线头、小脏污等      | 允许数量有限（AQL 限值）        |

> ❗培训中特别强调：
>
> * 一件衣服只有一个**很大的破洞** → **主要缺点（Major）**，**不是**严重缺点。
> * 严重缺点关键点在于：**安全风险 or 整批货风险**。

**★关键记忆点：**

* **严重 = 安全/整批风险；主要 = 单件无法销售；次要 = 勉强能卖但不满意。**
* 实际计数时：**主要和次要分开计数，不合并**（BS 自己的方式）。

## 五、尺寸测量 & 尺子校准

### 1. 尺子校准要求

| 项目     | 要求                                       |
| ------ | ---------------------------------------- |
| 校准周期   | 一般 **每 3 个月** 校准一次                       |
| 允许误差示例 | 20cm ±1mm；50cm ±1.5mm；100cm ±2mm（培训口径示意） |
| 注意对象   | 不仅 AQC 用的尺子要校准，也要**提醒工厂尺子**定期校准          |

### 2. 尺寸测量原则

| 内容     | 要点                                                                        |
| ------ | ------------------------------------------------------------------------- |
| 是否必须测  | 所有服装产品**必须做尺寸测量**                                                         |
| 品牌差异   | 不同品牌 **量法不同**（例如外套/夹克袖长：Jack & Jones vs 其他品牌）                             |
| 日常抽样   | 混色时，一般**每个尺码至少测 3 件**以上                                                   |
| 尺寸异常处理 | 若发现某色/某码超标：<br>• J&J / ONLY：按品牌具体尺寸要求执行<br>• 其他品牌：建议**额外多量 10–15 件**该码/颜色 |

### 3. J&J 尺寸 AQL 特例

| 情况          | 标准                            |
| ----------- | ----------------------------- |
| 正常首检        | 用 AQL 2.5（尺寸 + 做工）            |
| 复验（第一次不合格后） | 尺寸 & 做工改用 **AQL 1.5（更严格）**    |
| 尺寸抽样策略      | 抽多少件衣服就基本量多少件的尺寸，且分主部位 / 全部部位 |

**★关键记忆点：**

* **先分清自己做的是哪个品牌 → 再按对应量法 & 标准执行。**
* 尺子和验针机一样，**都要定期校准**，不只自己用的那把。

## 六、检验前准备资料清单（“开工前 Checklist”）

> ✅ 没有这些资料，**原则上不应该做自检**。

| 类别             | 具体内容                                                   | 备注                            |
| -------------- | ------------------------------------------------------ | ----------------------------- |
| 订单资料           | 最新的 Bestseller 原始订单（款号、订单号、数量、包装方式、交期、成分等）             | 业务可以遮住价格/交期等敏感信息，但**关键信息要完整** |
| 最新尺寸表          | 包含更新后的尺码规格                                             | 注意是否有最新版本                     |
| 样衣意见（Comments） | 买手批注、修改意见、工艺点说明，通常会有**锁扣编号**                           | 锁扣编号要与实物一致                    |
| 确认样（含锁扣）       | ★ 必须有锁扣（BS 锁扣或公司自有锁扣，且**不可拆**）<br>★ 标签上需能看出品牌、款号、寄出日期等 | 无锁扣样衣 → 原则上不接受                |
| 面料卡            | 正反面标识清楚，有签名/盖章                                         | 防止用反面、混布                      |
| 辅料卡            | 所有辅料齐全（纽扣、拉链、Logo、腰带等），中英文对照，签名/盖章                     | 不齐全 = 无法完整自检                  |
| 物理测试报告         | 含缩水、色牢度、克重等项目                                          | AQC 至少要看得懂“是否符合要求”            |

**★关键记忆点：**

* **“锁扣 + 标签 + 样衣照片” = 确认“这件就是最终标准样”。**
* 有任何资料拿不到 → 可以**私下找 QA 帮忙协调业务拿资料**。

## 七、抽箱 & 包装要求

### 1. 末期检验的抽箱原则

**思路图（简化版）：**

> 1️⃣ 先确保：**每个颜色 × 每个尺码 必须被抽到**
> 2️⃣ 再根据：**数量多的颜色/尺码 → 多抽；数量少的 → 少抽**
> 3️⃣ 再区分：
>   - 配比包装（Assort）：按箱数/配比抽即可
>   - 独码包装（Solid）：中码抽多一点，小/大码数量少但要保证有样本可量尺寸

**重要：**

* 📦 **抽箱人 = AQC 自己**，不可让工厂/他人代抽。
* 遇到破损箱：

  * 少量 → 在报告中备注，要求工厂更换。
  * 比例高 → 要求工厂查原因（受潮？箱子质量问题？）。

### 2. 包装方式结构表

| 维度   | 类型           | 说明                  |
| ---- | ------------ | ------------------- |
| 按形态  | 挂装           | 数量较少，按品牌要求执行        |
|      | 箱装           | 主流方式                |
| 箱装细分 | Loose（散装）    | 箱规统一，可能是独码/配比       |
|      | Assort（配比包装） | 大袋装 N 件 / 独立袋配比多种方式 |

### 3. 纸箱尺寸 & 结构

| 要点       | 内容                                               |
| -------- | ------------------------------------------------ |
| 基本规则     | 长度 L > 宽度 W ≥ 高度 H                               |
| Loose 订单 | 要有 **天地板**（上下垫板），因为丹麦仓库是**机器划圆开箱**，避免割伤衣物        |
| 尺寸范围     | 有最小 & 最大限制，按最新“包装手册”执行                           |
| 重量限制     | loose 和 配比最大重量上限不同（例如 loose ≈23kg，配比略低，具体以手册为准）  |
| 贴纸位置     | 正常：两个对角贴<br>箱子太矮 → 可部分贴到上面，只要**正/侧面都有可被扫码的位置**即可 |

### 4. 不要“死搬硬套箱规”

> ⚠ 典型错误：为了把西装/毛呢大衣塞进规定尺寸的箱子，**硬折 3–4 折** → 严重皱痕 → 客户索赔。

正确做法：

* 发现某类产品不适合按默认相规折叠 → **及时反馈给业务 & QA** → 申请特殊箱规。

**★关键记忆点：**

* AQC 要有判断力：**箱规服务于产品，而不是产品被箱规“折坏”。**
* 纸箱贴纸只要能让机器顺利扫码即可，不必追求“绝对对称距离几厘米”。

## 八、AQL 抽样 & 判定

### 1. 抽样 vs 标准的关系

| 概念         | 说明                                               |
| ---------- | ------------------------------------------------ |
| 抽样数量       | 同一批货，在不同 AQL（2.5 / 1.5 / 1.0）下 **抽样数相同**         |
| 不同 AQL 的差别 | 只体现在：**允许的缺点数量不同**：<br>AQL 值越小 → 标准越严 → 可接受缺点数越少 |

### 2. 培训中举的案例

> 订单数量：**10,001 件** → 抽样数：**315 件**

| 缺点类型          | 可接受数量                      |
| ------------- | -------------------------- |
| 严重缺点 Critical | ❌ 0 个                      |
| 主要缺点 Major    | ✅ ≤14 个                    |
| 次要缺点 Minor    | ✅ ≤14 个                    |
| 说明            | Major 和 Minor **各算各的，不合并** |

### 3. J&J / ONLY 复验标准

| 品牌                  | 首检                          | 复验          |
| ------------------- | --------------------------- | ----------- |
| Jack & Jones / ONLY | AQL 2.5                     | AQL 1.5（更严） |
| 尺寸部分                | 同上：尺寸抽样数不变，但**合格/不合格的判定更严** |             |

**★关键记忆点：**

* AQL 换成 1.5 不代表抽样更少，而是：**放行条件更苛刻**。
* 严重缺点一律 **0 容忍**。

## 九、防霉、湿度 & 干燥剂要求

### 1. 干燥房要求

| 项目   | 要求                                             |
| ---- | ---------------------------------------------- |
| 适用范围 | 缅甸、柬埔寨工厂 **必须有干燥房**；国内也逐步推广                    |
| 容量匹配 | 干燥房面积/容量要与工厂实际产能匹配                             |
| 湿度要求 | 干燥房湿度 **< 50%**（尤其雨季要严格控制）                     |
| 形式   | ① 独立干燥房（挂装/轨道式均可）<br>② 若整个后道均为空调+低湿环境，也可视为大干燥区 |

### 2. 干燥剂 & 防霉纸（Macropack）

| 产品     | 用途                                        | 场景           |
| ------ | ----------------------------------------- | ------------ |
| 防霉纸    | 放在**高风险材料**（真皮、PU 等）内部，释放成分抑制发霉（前提衣服本身已干） |              |
| 箱内干燥剂  | 放在纸箱内，控制箱内湿度                              | 将逐步替代传统硅胶干燥剂 |
| 集装箱干燥剂 | 挂在集装箱内，运输过程中吸湿                            | 工厂装柜时负责配置    |

> ✅ AQC 重点关注：是否按产品风险合理选择“防霉纸 / 干燥剂”，而不是“每单必须用同一种”。

**★关键记忆点：**

* **发霉 = 巨额损失** → 干燥房 + 干燥剂是硬措施。
* 湿衣服 + 防霉纸 = ❌，干燥是前提。

## 十、验针（减针）、物理测试 & 其他注意事项

### 1. 验针要求（针检）

| 项目     | 要求                                           |
| ------ | -------------------------------------------- |
| 标准     | 所有产品必须通过 **1.2mm**（训练中称“1.2 的减针”）金属检测        |
| 校准     | 验针机需定期用标准测试片校准                               |
| 不能过的处理 | 不能过时：**必须提前上报**（开发阶段 / 大货初期），与 QA / 买手讨论解决方案 |

> ❌ 不允许做法：工厂私下决定“不过减针就算了”，让 AQC 不写；一旦 BSQC 复验发现 → 货 & AQC 信誉都受损。

### 2. 物理测试

| 内容      | 要点                         |
| ------- | -------------------------- |
| 报告格式    | 可以用工厂/第三方自己的模板，不强制统一       |
| AQC能力要求 | **至少能看懂“是否合格”，能判断数据异常在哪里** |
| 例：克重公差  | 面料克重通常允许 **±5% 左右**（培训口径）  |

### 3. 其他典型问题

* 📷 **人台拍照：**

  * 外套类：人台上最好套一件薄内搭（T 恤/衬衫），不要“光膀子穿外套”。
  * Jack & Jones / JJXX：**必须用指定人台**，没有就写“不适用”，不要用其他品牌人台代替。
  * 其他品牌：有 BS 人台就用；没有则可不拍人体照。

* 📄 **拷贝纸（薄纸）使用：**

  * 无**硬性要求**一件一张。
  * 但对于有纽扣/腰带易压出痕迹的款，建议用拷贝纸隔开或包住按扣、扣子。

## 十一、AQC 自检报告 & 系统操作要点

| 项目   | 要求                                       |
| ---- | ---------------------------------------- |
| 系统   | 使用 Bestseller 的在线系统（类似 Karma/Qualma 手机端） |
| 结论选项 | 只有 **合格 / 不合格**，**无“待定”**选项              |
| 填写原则 | 必须**真实、清楚**地反映问题，并尽量给出合理建议               |
| 尺寸录入 | 上传前务必自查，无误再提交，因为**提交后不能修改/撤回**           |
| 照片   | 尽量清晰、完整地反映问题/整体效果；避免不规范穿法/人台问题           |

## 十二、产前会（PP Meeting）与质量手册

### 1. 产前会参加人员

| 类型      | 示例                  |
| ------- | ------------------- |
| 工厂生产管理层 | 生产经理、车间主管等          |
| 品质相关    | 工厂 QC / QA、面辅料验收负责人 |
| 外贸公司    | 业务、跟单等              |
| 必要时     | BS QA、技术支持人员        |

### 2. 产前会两个核心目的

1. ✅ **回顾准备工作是否完成**

   * 面料/辅料是否已检验并合格
   * 样衣是否已确认并有定版
   * 工艺/样板是否到位

2. ✅ **明确品质要求**

   * 外观、做工、尺寸、包装、验针、物理测试等**所有质量要求**讲清楚

### 3. 质量相关手册

培训中提到：公司有多本手册，如：

* 质量手册
* 工艺/特良手册
* 安全（儿童安全等）手册
* 验针/金属检测手册
* 包装/纸箱规范手册 等

**★关键记忆点：**

* 不记得标准时：**先翻手册，再问 QA**。
* 比如：布料回潮放松时间到底是 24h 还是 48h → 以质量手册为准。

## 十三、整体“自检流程”小图示（复习用）

```text
【接单】 
   ↓（向业务/工厂要齐全资料）
【确认资料】
   - 原始订单
   - 最新尺寸表
   - 样衣 + 锁扣
   - 面料卡、辅料卡
   - 物理测试
   ↓
【检前准备】
   - 检查工具校准（尺子、验针机、湿度计）
   - 了解包装方式 & 箱规
   ↓
【抽箱】
   - AQC 自己随机抽
   - 覆盖所有 色 × 码
   - 数量多 → 多抽
   ↓
【现场检验】
   - 外观做工（按 缺点分类：Critical / Major / Minor）
   - 尺寸测量（按品牌要求）
   - 验针、称重、湿度、防霉措施
   ↓
【记录 & 拍照】
   - 问题拍清楚
   - 数据录入系统
   ↓
【出结论】
   - AQL 判定：合格 / 不合格
   - 重要问题及时同步工厂 + 自家领导 + BS QA
```
